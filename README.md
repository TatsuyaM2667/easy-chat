# easy-chat
## 事前準備
### 1. clone this project
```
$ git clone git@github.com:kappa0923/easy-chat.git
```

### 2. install node
```
$ brew install node
or
https://nodejs.org/ja/
```

### 3. install firebase-tools
```
$ npm install -g firebase-tools
```

#### FYI
https://firebase.google.com/docs/cli/?hl=ja

### 4. check firebase-tools version
```
$ firebase --version
3.15.0 or newer
```
